## 🧪 3分钟快速测试

### 测试1: 富文本编辑器（推荐首先测试）

1. 打开浏览器访问 http://localhost:3100
2. 登录或注册账号
3. 点击"创建笔记"，选择类型为"富文本"
4. 输入一些文字，例如：
   ```
   这是一段测试文字，用于验证选择功能是否正常工作。
   ```
5. **用鼠标选中"测试文字"这几个字**
6. **观察右侧 AI 助手输入框**
7. 应该看到：
   ```
   📋 已选择富文本内容: "测试文字..."          [清除]
   ```

**如果看到上面的提示框 → ✅ 功能正常！**

---

### 测试2: 思维导图

1. 点击"创建笔记"，选择类型为"思维导图"
2. 双击画布添加中心主题
3. 按 Tab 键添加子节点
4. **按住 Shift 键，点击选中2-3个节点**
5. **观察工具栏**，应该显示"发送 N 个节点"按钮
6. 点击"发送"按钮
7. 观察 AI 助手是否显示节点结构

**如果按钮高亮并可点击 → ✅ 功能正常！**

---

### 测试3: 代码编辑器

1. 点击"创建笔记"，选择类型为"代码编辑器"
2. 选择语言：JavaScript
3. 输入代码：
   ```javascript
   function hello() {
     console.log("Hello World");
   }
   ```
4. **选中 `console.log("Hello World");` 这行**
5. 观察 AI 助手是否显示选择提示

**如果显示选择提示 → ✅ 功能正常！**

---

## ❓ 如果功能不工作

### 检查1: 打开浏览器控制台
按 `F12` 键，切换到 Console 标签，查看是否有红色错误

### 检查2: 强制刷新
按 `Ctrl + Shift + R` (Windows) 或 `Cmd + Shift + R` (Mac) 强制刷新页面

### 检查3: 确认编辑器类型
确保您使用的是以下编辑器之一：
- ✅ Markdown
- ✅ 富文本
- ✅ 代码编辑器 (Monaco)
- ✅ 思维导图
- ✅ DrawIO
