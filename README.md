# AiNote - AI智能笔记应用

一款具有AI辅助功能的现代化笔记应用，参考有道云笔记的简洁高效设计理念。

## 功能特性

### 核心功能
- ✅ 笔记管理：创建、编辑、删除、搜索笔记
- ✅ 分类管理：多分类支持，便于笔记组织
- ✅ 标签系统：灵活的标签管理
- ✅ 收藏功能：快速访问重要笔记
- ✅ 自动保存：防止数据丢失

### AI助手功能
- ✅ 智能对话：自然语言交互
- ✅ 内容生成：根据提示生成新内容
- ✅ 改写润色：优化文字表达
- ✅ 摘要提取：快速提取核心内容
- ✅ 关键词生成：自动生成内容关键词
- ✅ 翻译功能：中英文互译
- ✅ 语法修正：检查并修正错误
- ✅ 内容扩展：丰富文章内容

### 模型管理
- ✅ 多模型配置：支持多个API配置
- ✅ 参数调整：温度、最大Token、Top P等
- ✅ 连接测试：验证API配置
- ✅ 使用统计：Token使用量监控
- ✅ 调用日志：详细的使用记录

### 用户体验
- ✅ 响应式设计：适配不同屏幕尺寸
- ✅ 主题切换：亮色/暗色模式
- ✅ 快捷键支持：提高操作效率
- ✅ 流畅动画：优雅的交互体验

## 技术栈

- **前端框架**: React 18 + TypeScript + Vite
- **UI组件库**: Ant Design 5
- **状态管理**: Zustand
- **数据存储**: IndexedDB (Dexie.js)
- **样式方案**: Styled Components + CSS
- **AI集成**: 智谱AI GML-4.7 API

## 配置说明

### AI模型配置

1. 打开应用后，进入"模型管理"页面
2. 点击"添加配置"
3. 填写以下信息：
   - **配置名称**: 例如"GML-4.7"
   - **API密钥**: 从智谱AI开放平台获取
   - **API端点**: `https://open.bigmodel.cn/api/paas/v4/chat/completions`
   - **模型名称**: `glm-4`
   - **温度**: 0.7（控制随机性，0-2）
   - **最大Token**: 2000
   - **Top P**: 0.9
4. 点击"测试"验证配置
5. 启用配置

## 使用说明

### 创建笔记

1. 点击左侧边栏的"新建笔记"按钮
2. 在编辑器中输入标题和内容
3. 内容会自动保存

### 使用AI助手

1. 点击顶部的"AI助手"按钮打开侧边栏
2. 选择快捷操作或直接输入问题
3. AI会实时返回响应

### 管理笔记

- 使用搜索框快速查找笔记
- 点击星标收藏重要笔记
- 创建分类组织笔记
- 添加标签方便筛选

## 快捷键

- `Ctrl/Cmd + N`: 新建笔记
- `Ctrl/Cmd + S`: 保存笔记
- `Ctrl/Cmd + F`: 搜索笔记
- `Ctrl/Cmd + Enter`: 发送AI消息

## 注意事项

1. **API密钥安全**: 请妥善保管您的API密钥，不要泄露
2. **数据存储**: 所有数据存储在本地浏览器中，清除浏览器数据会丢失笔记
3. **浏览器兼容**: 建议使用最新版本的Chrome、Edge或Safari
4. **网络要求**: 使用AI功能需要网络连接
